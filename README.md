# @yangss/utils
A collection of common JavaScript / TypeScript utils

- Tree-shakable ESM
- Fully typed
- Type utilities
- Zero dependence

## Install
```sh
# npm
npm i @yangss/utils
# yarn
yarn add @yangss/utils
```
